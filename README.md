# Лабораторна робота №3

## Операції зі стрічками. Застосування класів-колекцій і класів утиліт

## Завдання
Проаналізувати стрічку, яка задана аргументом командного рядка "пільги=yes&обєм=1600", що є способом задання вихідних даних
у задачі обчислення розміру транспортного податку. Якщо є пільги і об'єм понад 1500см.куб., то податок не сплачують.
Інші умови довільно задаються студентом, а також ставка за кожні 100см.куб. робочого об'єму двигуна автомобіля.
